# おみくじアプリ 
![アプリ画面](omikuji.JPEG)

Pythonの**tkinter**を使って作成したシンプルなおみくじアプリです。 

ボタンを押すとランダムで運勢が表示されます。

## 概要
このアプリは、PythonのGUIライブラリ **tkinter**を使って作ったおみくじアプリです。  

「おみくじを引く」ボタンを押すと、ランダムで運勢とコメントが表示されます。

## 機能

- ボタンを押しておみくじを引く
- 運勢ごとに確率を設定
- 運勢ごとにコメント表示
- 「超大吉」は金色で表示

## おみくじの確率

| 運勢 | 確率 |
|-----|------|
| 超大吉 | 5% |
| 大吉 | 10% |
| 中吉 | 30% |
| 小吉 | 40% |
| 凶 | 20% |
